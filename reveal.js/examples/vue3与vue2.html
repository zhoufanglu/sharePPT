三、API的变化

  • VUE2 - options Api
    • 一个功能分散到了不同的vue的配置项中
  • VUE3 - composition Api
    • 代码根据逻辑功能来组织的，一个功能的所有api会放在一起（高内聚，低耦合）

options Api

缺点：

  • 光标反复横跳
  • mixin命名重复
  • this的多次使用
  • 功能多了之后不易维护
